# Amazonsales_power-bi
📊 Amazon Sales Analytics Dashboard (Power BI)
📌 Project Overview

This project presents an interactive Amazon Sales Dashboard built using Power BI. It provides deep insights into sales performance, customer behavior, product trends, and future forecasting.

The dashboard is designed to help businesses make data-driven decisions by analyzing key metrics such as sales, profit, customer segmentation, and category performance.
